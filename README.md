## ⚙️ Como executar o projeto

Este projeto é divido em duas partes:
1. App react-native
2. Api em nodejs

```bash
# Abra um novo terminal e execute o comando:
$ sh start-api.sh

# Abra um novo terminal e acesse a pasta tasks-app
$ cd app

# Execute o comando:
$ npx react-native run-android
```
